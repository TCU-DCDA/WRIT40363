Mobile Navigation Test

Resize your browser window to mobile size (768px or smaller) to test the hamburger menu.

The hamburger icon should appear and be clickable to show/hide the navigation menu.

Instructions:

  1. Resize browser window to mobile size (768px width or smaller)
  2. You should see a hamburger icon (three lines) in the top navigation
  3. Click the hamburger icon to open the mobile menu
  4. Click it again to close the menu
  5. Click outside the menu to close it
  6. Check browser console for debug messages
