Course Assignments

Laboratory exercises, discussions, and projects designed to build your multimedia authoring expertise

📋 Assessment Philosophy

"We focus on meaningful feedback and self-reflection rather than traditional point-based grading."

Your progress will be evaluated through detailed feedback categorized as:

Exceeds Expectations

Demonstrates mastery beyond course requirements

Meets Expectations

Successfully fulfills course standards

Needs Improvement

Requires additional attention and refinement

🧪 Laboratory Exercises (6 Total)

Hands-on technical challenges designed to build your multimedia authoring skills progressively throughout the semester.

Laboratory Exercise 1

HTML Fundamentals

Objective: Master the basic syntax and structure of HTML markup

Skills Developed: Document structure, semantic elements, accessibility principles

Due: End of Week 1 (Sunday 11:30 PM)

Laboratory Exercise 2

CSS Styling

Objective: Apply visual styling and layout principles using CSS

Skills Developed: Selectors, properties, responsive design, visual hierarchy

Due: End of Week 3 (Sunday 11:30 PM)

Laboratory Exercise 3

JavaScript Logic

Objective: Implement programming logic with variables, conditionals, and functions

Skills Developed: Problem-solving, algorithmic thinking, code organization

Due: End of Week 5 (Sunday 11:30 PM)

Laboratory Exercise 4

Data Collection

Objective: Gather and process data using web scraping and API techniques

Skills Developed: Data analysis, API integration, information processing

Due: End of Week 6 (Sunday 11:30 PM)

Laboratory Exercise 5

DOM Manipulation

Objective: Dynamically modify web page content using JavaScript

Skills Developed: Interactive design, user experience, dynamic content

Due: End of Week 8 (Sunday 11:30 PM)

Laboratory Exercise 6

Testing & Debugging

Objective: Implement testing strategies and troubleshooting techniques

Skills Developed: Quality assurance, problem diagnosis, code optimization

Due: End of Week 9 (Sunday 11:30 PM)

💬 Discussion Forums (3 Total)

Thoughtful discourse on multimedia authoring theory, practice, and cultural implications of technological innovation.

Discussion Forum 1

Multimedia Foundations

Topic: The intersection of traditional media and digital multimedia technologies

Format: Online forum engagement with thoughtful responses to peers

Due: End of Week 2 (Sunday 11:30 PM)

Discussion Forum 2

Design & Society

Topic: Ethical considerations in multimedia design and user experience

Format: Critical analysis of readings with peer dialogue

Due: End of Week 7 (Sunday 11:30 PM)

Discussion Forum 3

Future Technologies

Topic: Emerging multimedia technologies and their impact on content creation

Format: Synthesis of course concepts with forward-looking analysis

Due: End of Week 10 (Sunday 11:30 PM)

📝 Self-Reflection Essays (3 Total)

Personal assessment essays documenting your learning journey and academic growth throughout the course.

Assessment Dimensions

  • 💻Coding Competency: HTML, CSS, and JavaScript mastery
  • 👥Audience Engagement: Accessibility and User Experience (UX)
  • 📐Project Design: Planning and architectural thinking
  • 🤝Collaboration: Working effectively with fellow students

Reflection Schedule

Self-Reflection 1

Focus: Initial goals and expectations

Due: End of Week 1

Self-Reflection 2 (Midterm)

Focus: Progress assessment and grade self-evaluation

Due: End of Week 7

Self-Reflection 3 (Final)

Focus: Complete learning journey and final grade argument

Due: Finals Week

🚀 Final Project

The Capstone Project

The Final Project represents the culmination of your course experience—a comprehensive demonstration of multimedia authoring skills that synthesizes all acquired knowledge into a meaningful interactive application.

Project Structure

  • 👤Individual Projects: Personal exploration of multimedia concepts
  • 👥Collaborative Projects: Team-based investigation of complex design challenges

Technical Requirements

  • 📱Responsive web design using HTML, CSS, and JavaScript
  • 🗄️Database integration for dynamic content
  • 🎨User-centered design principles
  • ♿Accessibility and usability considerations
  • 📖Clear documentation and code comments

Project Milestones

Project Proposal

Week 11: Detailed project plan and objectives

Dataset Biography

Week 12: Analysis of data sources and ethical considerations

Group Contract

Week 12: Collaboration agreements (team projects only)

Final Submission

Week 16: Complete project with documentation

Grand Exhibition

Finals Week: Public presentation of innovations

📋 Assignment Policies

Submission Guidelines

  • 📤All assignments submitted through course portal (D2L)
  • 📁Include descriptive file names and clear organization
  • 🤖Document any AI assistance used in code comments
  • 💾Maintain backup copies of all work

Late Work Protocol

While no point deductions apply, patterns of late submissions may compromise your final self-assessment. Contact instructors before deadlines to request extensions.

Collaboration Policy

Discussion and peer assistance encouraged. For individual assignments, submit your own work. For group projects, clearly document individual contributions.

Technical Support

Utilize the Center for Digital Expression (Scharbauer 2003) for project consultation. Contact instructors or IT support at least one week before assignment deadlines for technical difficulties.
